# AION Party 1.0

`list.txt`에 있는 8명의 본캐/부캐 정보를 읽어서 4개 공격대, 8개 파티를 자동 편성하고 결과를 `party_result.txt`로 저장합니다.

## list.txt 형식

```text
캐릭터(본, 직업, 전투력) / 캐릭터(부, 직업, 전투력) / 캐릭터(부, 직업, 전투력) / 캐릭터(부, 직업, 전투력)
```

- 총 8줄이어야 합니다.
- 한 줄은 한 유저입니다.
- 각 줄은 본캐 1개, 부캐 3개여야 합니다.
- 직업은 `수호성`, `검성`, `살성`, `호법성`, `치유성`, `마도성`, `정령성`, `궁성` 중 하나여야 합니다.
- 한 캐릭터는 전체 결과에서 한 번만 사용됩니다.
- 같은 유저의 캐릭터는 같은 공격대 안에 중복 배치되지 않습니다.
- UI에서 `본1 / 부1~부3` 구성을 선택할 수 있습니다. 기본값은 `본1 / 부3`입니다.
- 본캐가 `치유성` 또는 `호법성`이면 전투력 평균이 과하게 벌어지지 않는 범위에서 강한 부캐를 우선 배치합니다.

## 실행

```powershell
AION.Party.exe
```

또는 배포 파일을 만든 뒤 실행합니다.

```powershell
.\release.ps1 make  
.\release.ps1 clean
```

결과 파일:

```text
party_result.txt
```
